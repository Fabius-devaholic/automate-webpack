# automate-webpack


## Install 
```
yarn
```

## Run dev server
```
yarn dev
```

## Build files
```
yarn build
```

## Project Structure
```
build/      # webpack configuration files
src/
|
|- assets/  # will be compiled by webpack
|- scripts/ # script files
|- styles/  # style files
static/     # static files
...
```
